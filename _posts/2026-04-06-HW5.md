## Plot 1: Top 20 States by Bigfoot Sightings

<vegachart schema-url="{{ site.baseurl }}/assets/json/plot1.json" style="width: 100%"></vegachart>

<br>

This bar chart displays the top 20 U.S. states with the highest number of Bigfoot sightings recorded in the BFRO dataset. Each bar represents a state, ranked from most to fewest sightings. I used a horizontal bar chart so state names are readable without rotation. The x-axis encodes the count of sightings as a quantitative variable, and the y-axis encodes state names as a nominal variable sorted by count. I applied a sequential "teals" color scheme mapped to the count variable — darker teal indicates more sightings — which reinforces the magnitude differences already shown by bar length. For data transformations, I used pandas `value_counts()` to aggregate sightings per state and then selected only the top 20 rows to keep the chart readable. No colormap is strictly necessary here since bar length already encodes count, but the color gradient adds a secondary visual cue that makes high-count states stand out more immediately.

## Plot 2: Bigfoot Sightings by State and Classification

<vegachart schema-url="{{ site.baseurl }}/assets/json/plot2.json" style="width: 100%"></vegachart>

<br>

This choropleth map visualizes the number of Bigfoot sightings across U.S. states, colored by sighting intensity. The map uses a sequential "blues" color scheme — gray indicates zero sightings for that classification, light blue indicates fewer sightings, and dark blue indicates higher sighting counts — making it immediately intuitive to spot hotspot states. The positional encoding is geographic, with each state shape representing a nominal spatial unit, and the color encodes the quantitative count of sightings. For data transformations, I grouped the dataset by state and classification, counted sightings for each combination, mapped state names to their numeric FIPS IDs (required for the TopoJSON map), and pivoted the data into separate columns for Class A, B, and C to enable dynamic switching. The interactivity consists of a dropdown menu that filters the map by classification type (Class A, B, or C). This is valuable because it lets the viewer compare geographic patterns across classification types — for example, seeing whether Class A direct sightings cluster differently than Class B indirect evidence reports — which would be impossible to see clearly if all classifications were combined. Without this interactivity, overlapping classifications would obscure geographic patterns, making it impossible to isolate and compare where each type of sighting is most commonly reported across the country.
