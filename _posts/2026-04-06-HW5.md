## Plot 1: Top 20 States by Bigfoot Sightings

<vegachart schema-url="{{ site.baseurl }}/assets/json/plot1.json" style="width: 100%"></vegachart>

This bar chart displays the top 20 U.S. states with the highest number of Bigfoot sightings recorded in the BFRO dataset. Each bar represents a state, ranked from most to fewest sightings. I used a horizontal bar chart so state names are readable without rotation. The x-axis encodes the count of sightings as a quantitative variable, and the y-axis encodes state names as a nominal variable sorted by count. I applied a sequential "teals" color scheme mapped to the count variable — darker teal indicates more sightings — which reinforces the magnitude differences already shown by bar length. For data transformations, I used pandas `value_counts()` to aggregate sightings per state and then selected only the top 20 rows to keep the chart readable. No colormap is strictly necessary here since bar length already encodes count, but the color gradient adds a secondary visual cue that makes high-count states stand out more immediately.

## Plot 2: Bigfoot Sightings by Location, Season & Classification

<vegachart schema-url="{{ site.baseurl }}/assets/json/plot2.json" style="width: 100%"></vegachart>

This scatter plot maps Bigfoot sightings across the continental United States using latitude and longitude as the x and y positional encodings (both quantitative). Each point represents one reported sighting, and the color encodes the classification of the report as a nominal variable — Class A (direct sightings), Class B (indirect evidence), and Class C (second-hand reports) — using the "set1" color scheme, which provides clearly distinct hues for categorical differentiation. For data transformations, I dropped rows with missing values in the season, classification, latitude, and longitude columns, filtered to only the four main seasons, and filtered coordinates to the continental US bounding box (latitude 24–50, longitude -125 to -66) to remove erroneous outlier points. The interactivity consists of a dropdown menu that filters the map by season (Spring, Summer, Fall, Winter). This is valuable because it lets the viewer explore whether sighting patterns shift geographically across seasons, which would be impossible to see clearly if all seasons were shown at once.
